社会招聘

高级财务经理 上海

工作职责:

1. 协助分管副总经理制定公司战略，从财务、资本、风险控制、政策、投资等方向提出前瞻性的预警和措施；并主持集团公司财务战略规划的制定；
2. 负责集团税收筹划及各成员公司利润管理；
3. 负责集团各BU经营分析体系完善及KPI数据及时提供；
4. 负责定期和各BU,部门开展经营分析活动,主导公司成本降低；
5. 负责建立和完善公司财务制度及符合公司实际情况的财务核算体系和财务监控体系，进行有效的内部财务控制和企业评估；
6. 制定集团公司资金运营计划，资金管理报告和预、决算，监督、检查各部门及项目公司预算管理和资金计划的执行情况，并根据实际情况出具调整意见；制定集团公司的会计核算方案；
7. 负责对公司及项目公司的财务状况和经营成果进行分析，对资金运用、现金流量、财务成本、盈利转化与退出等进行专题分析，并向上级领导提交相关财务分析报告；
8. 完成上级临时交办的其他任务。

岗位要求:

1. 集团化公司管理经验，多年财务管理经验; 8年以上财务工作经验；
2. 英语流利；
3. 上市公司工作经验优先考虑；
4. 抗压能力强。
投资并购总监 上海

工作职责：

1. 根据公司发展战略和业务要求，寻找及筛选投资并购项目；
2. 对目标企业（项目）进行资料收集、市场调查、市场预测、投资可行性研究；
3. 负责投资方向和项目价值评估分析，撰写投资价值分析报告、投资立项报告和投资方案；
4. 对投资项目组织和实施尽职调查，财务模型的设立与评估，参与合同谈判并提供专业意见；
5. 对已投资行业或企业进行研究分析，发掘价值增值机会；
6. 参与收购后的部分整合工作。

岗位要求：

1. 本科及以上学历，经济、金融、财务、及法律专业背景；
2. 具有5年以上相关工作经验，须具有投资并购项目管理及运作经验；
3. 具备投资管理所需的财务及金融、法律相关专业知识，收悉收购与兼并知识与技能；
4. 熟悉资本运营法律法规、财务规范及投资并购流程与交易结构方案设计；
5. 具有较强的价值分析和评估能力，具有较强的项目组织和议价能力及项目运作策划能力；
6. 具备扎实的宏观经济和产业经济理论基础，深入了解国内外资本市场、产业市场的发展轨迹和趋势。
投资并购法务总监 上海

工作职责：

1. 协助董秘完成上市公司并购重组、股权基金等资本市场业务；
2. 主要工作包括但不限于并购目标的帅选，法律尽职调查，起草项目交易文件、法律意见书、律师工作报告、专项法律意见，解答客户法律问题等；
3. 其它与职位相关的工作。

岗位要求：

1. 国内知名法学院本科及研究生学历，有海外留学经历者优先，已取得律师执业证；
2. 须具有IPO、新三板等资本市场项目的实务操作经验，了解并购，或有诉讼经验三年；
3. 工作认真、细致、踏实、稳重，具有团队合作精神，良好的沟通能力；
4. 具有较深的法学理论功底、较好的语言交流及书面表达能力；
5. 英语佳，CET6以上，托福、雅思、高级口译成绩较好者优先；
6. 具有吃苦耐劳、细致认真的品质和敬业精神，性格随和。
投资经理 上海

工作职责：

1. 根据公司所投资领域的产业方向，独立完成所分配项目所述的行业、领域的深度研究（数据采集，分析，报告）；
2. 发掘所关注行业、领域的投资机会；
3. 能够带领团队，完成项目立项报告、项目尽职调查、投资协议谈判等工作；
4. 跟踪相关行业、领域的国内外主要企业的动态，在公司相关资源的配合下，形成相对紧密的业务合作网络。

岗位要求：

1. 具备完善的数据采集渠道、分析框架、研究方法；
2. 具有缜密的思维和较强的逻辑分析能力；
3. 熟悉相关行业领域国内外、产业链上下游相关企业情况，能够准确地判断和捕捉潜在的投资机会；
4. 具备两年以上VC、PE投资经理工作经历者优先或者上市公司投资相关工作经验2年以上；
5. 有相关行业内企业人脉资源者更佳；
6. 具有经济或理工科背景，能够对于相关技术领域的研究方向进行很好的理解和分析；
7. 复合教育背景优先，具有注册会计师资质者尤佳。
经营分析专员 滁州

工作职责：

1. 参与公司整体预算编制，开展日常追踪；
2. 公司日常经营数据的整理和分析，出具公司经营分析报告；
3. 配合制定各部门KPI，追踪分析并配合人事完成考核工作；
4. 配合制定各部门降低成本项目，追踪项目完成情况；
5. 配合公司制定阿米巴管理方案，跟踪分析各阿米巴经营体的经营结果；
6. 对公司整体财务情况以及各业务情况进行各项财务指标的综合分析和评价，提出财务目标改进建议，协助业务决策；
7. 领导安排的其他工作。

岗位要求：

1. 本科及以上学历，财务管理及会计学等财务相关专业优先考虑；
2. 3年以上大型企业财务分析、预算管理工作经验优先；
3. 工作认真负责、责任心强，具备独立思考能力，能承受一定压力；
4. 对数据敏感性强，能够熟练运用excel、PPT，具有较强的逻辑分析能力；
5. 具备优秀的沟通协调、组织和学历能力；
6. 职业素养高，保密能力强。
海外销售总监 东莞

工作职责：

1. 负责公司海外业务日常管理和开拓工作；
2. 负责海外市场业务拓展、品牌推广、策划、销售计划的落实完成工作；
3. 负责海外市场的销售目标的划分、制定、完成；
4. 依据公司经营目标制定进入海外市场策略，以及制定进入新市场的具体措施；
5. 完成上级领导安排的其他工作。

岗位要求：

1. 大学本科以上学历；
2. 8年以上销售经验，其中3年以上手机/平板保护类产品行业销售经验；
3. 熟悉国际市场，有固定的客户资源；
4. 优秀的组织能力、表达能力、分析能力和执行力；
5. 良好的学习创新和团队管理能力；
6. 可适应不定期国外出差；
7. 英语口语流利，有海外留学或者工作经验者优先；
8. 踏实敬业，沟通能力强，抗压能力强。
